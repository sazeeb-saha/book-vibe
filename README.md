# 📚 Book Vibe

Book Vibe is a simple and modern React-based book tracking application where users can mark books as **Read** or add them to a **Wishlist**. It uses localStorage to persist data and provides sorting features for better user experience.

---

## 🚀 Features

- 📖 Mark books as "Read"
- ❤️ Add books to Wishlist
- 🔄 Prevent duplicate entries
- 📊 Sort books by:
  - Pages
  - Rating
- 💾 Persistent storage using LocalStorage
- ⚡ Fast and responsive UI
- 🔔 Toast notifications for user actions

---

## 🛠️ Tech Stack

- React.js
- React Context API
- Tailwind CSS
- React Toastify
- LocalStorage (for data persistence)
